# 🖱️ Cursor Trail Animation

An interactive **cursor trail animation** that creates smooth circular patterns following the mouse movement.

This project demonstrates how to create visually appealing cursor effects using **HTML, CSS, and JavaScript**.

---

## ✨ Features

* 🖱️ Smooth cursor trail animation
* 🎨 Minimal and clean design
* ⚡ Lightweight JavaScript effect
* 💻 Works in any modern browser

---

## 🚀 Demo

Move your mouse across the screen and watch the animation follow the cursor.

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
